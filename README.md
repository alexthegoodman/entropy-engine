# Entropy Engine

Powering a unified OS inspired by MCP.

- Quadtree landscapes
- Point lighting
- Game behaviors
- Kinematic animations
- Rendered images and videos
- Rendered text with fonts
- Screen capture
- Vector animations
- Video export
- GLB Import
- and more!

## Run

Example Game:
- `cargo run --bin game --release`

Level Editor: 
- `cargo run --bin entropy-engine --release`
